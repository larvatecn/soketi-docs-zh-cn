# soketi 中文文档

Soketi is your simple, fast, and resilient open-source WebSockets server.

## 如何在本地编辑和预览该网站

本站基于 [VitePress](https://github.com/vuejs/vitepress) 和 [@vue/theme](https://github.com/vuejs/vue-theme) 建立。网站内容以 Markdown 格式书写，位于 `src` 文件夹中。

```sh
pnpm i
pnpm run dev
```

本项目要求 Node.js 为 `v14.0.0` 或更高版本，因为我们在代码中使用了新的 JavaScript 功能，例如可选链。

## 如何参与贡献

有劳访问我们的 [GITHUB](https://github.com/larvatecn/soketi-docs-zh-cn) 了解相关注意事项。
